Class: ChannelPublicView

ChannelPublicView()

new ChannelPublicView()

This is the page that an user will be redirected to if the user is not subscribed to that channel. It only shows general info for this channel. User can join that channel by clicking on the button.
Properties:
Name Type Description
user_id number A unique number identifying the user.
channel Object An object describing a specific channel.
user_channel Object An array of channels that the user is subscribed to.
task Object An object containing all tasks related to this channel.
channelActivities array An array of all completed activities by a user.
subscribed_users Object An object containing all users subscribed to a particular channel.
subscribe function A function allowing the user to subscribe to a particular channel.
