Class: PerformancePage

PerformancePage()

new PerformancePage()

This is the component that shows analytical stats for an user. It contains number of task the user has done in the past day, week and month, with a chart. It also shows the user's score in each channel.
Properties:
Name Type Description
userActivities array An array containing all of the user's finished task activities.
subscribed_channels Set A set containing all of the user's subscribed channels.
channels Object An object containing all the channels.
userId number A unique number identifying the user.
