Class: TaskDetail

TaskDetail()

new TaskDetail()

This is the component that shows all the information of a task in that channel. It also allows the user to enroll/drop that task.
Properties:
Name Type Description
title string The title of the task.
period number A type encoding indicating the day, week, or month period of a task.
pattern number A value indicating the numbere of times a user can complete a task within a given period.
points number A value indicating the points associated with the task.
enrolled boolean A boolean value indicating whether the user is enrolled in a task or not.
task_id number A unique number identifying the task.
user_id number A unique number identifying the user.
enroll function A function allowing the user to subscribe to a task.
drop function A function allowing the user to unsubcribe from a task.
newActivity function A function allowing a user to create a new task.
